HangManGame adalah program bahasa Java yang memilki konsep mirip dengan game Hangman pada umumnya. Game ini menggunakan JPanel Java sebagai Interface.

Cara Main :
1. Sebuah kata dalam bahasa Inggris akan ditampilkan pada Interface. Seluruh huruf pada kata tersebut disensor dengan bintang (*).
2. User memiliki "nyawa" atau kesempatan menebak sebanyak huruf pada kata tersebut dan diminta untuk menebak satu persatu hurufnya.
3. Jika User benar menebak, maka bintang (*) akan berubah menjadi huruf yang sudah ditebak. Jika User salah menebak, maka nyawa akan dikurangi 1.
4. Jika User berhasil menebak seluruh huruf, maka game selesai dan User menang.
4. Jika nyawa User habis, maka game akan selesai dan User kalah.
    Contoh : User mendapat kata "fan". Tampilan awal akan berupa "***" dan nyawa = 3. Jika User menebak huruf A, maka tampilan berubah menjadi "*a*" dan nyawa = 3. Selanjutnya jika User menebak huruf X, maka tampilan tetap "*a*" dan nyawa berkurang satu, sehingga menjadi nyawwa = 2.
